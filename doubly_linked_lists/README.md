This project focuses on understanding and implementing doubly linked lists in C.

*Tasks Summary*

0-rint_dlistint.c

Write a function that prints all the elements of a dlistint_t list.
Returns the number of nodes.
--------------------------------------

1-dlistint_len.c

Write a function that returns the number of elements in a dlistint_t list.
--------------------------------------

2-add_dnodeint.c

Write a function that adds a new node at the start of the list.
Returns the address of the new node or NULL on failure.
--------------------------------------

3-add_dnodeint_end.c

Write a function that adds a new node at the end of the list.
Returns the address of the new element or NULL on failure.
--------------------------------------

4-free_dlistint.c

Write a function that frees an entire dlistint_t list.
--------------------------------------

5-get_dnodeint.c

Write a function that returns the node at a specific index in the list.
If the index does not exist, return NULL.
--------------------------------------

6-sum_dlistint.c

Write a function that returns the sum of all the data (n) in the list.
If the list is empty, return 0.
--------------------------------------

7-insert_dnodeint.c

Write a function that inserts a node at a given position.
If the index is out of range, return NULL.
--------------------------------------

8-delete_dnodeint.c

Write a function that deletes a node at a specific index.
Return 1 on success, or -1 on failure.
