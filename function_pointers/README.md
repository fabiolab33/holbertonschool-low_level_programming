The project explore the use of fuction pointers in C.

Files:

0-print_name.c
A fuction thats prints a name using a function pointer.

1-array_iterator.c
Fuction that executes another function on each element of an array.

2-int_index.c
A function thats searches an array using a function pointer as comparator.

3-calc.h
Header with prototypes and operator
3-get_op_func.c
Returns the correct function pointer based on the operator.
3-main.c
Handles user input and calls the proper operation.
3-op_functions.c
Contains 'op_add', 'op_sub', 'op_mul', 'op_div' and 'op_mod'.

This ones (3-) implements a calculator that performs operations selected at runtime using function pointers.
