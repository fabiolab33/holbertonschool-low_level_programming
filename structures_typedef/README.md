This project focuses in learning how to use structures and typedef in C.

Files:

0-dog.h
Contains the definition of 'struct dog' and 'dog_t' typedef.

1-init_dog.c
Initializes a 'struct dog'.

2-print_dog.c
Prints the values of a 'struct dog''

3-dog_typedef.c
Defines 'dog_t' as a new name for 'struct dog'.

4-new_dog.c
Creates a new dog with copies name and owner.

5-free_dog.c
Frees memory allocated for a dog.
