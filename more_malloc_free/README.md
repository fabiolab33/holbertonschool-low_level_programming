The project focuses on dynamic memory allocation in C.

Files:

0-malloc_checked.c
Allocates memory using malloc and exist with status 98 if allocation fails.


1-string_nconcat.c
Concatenates two strings, copying at most n bytes from the second string into newly allocated memory.

2-calloc.c
Allocates memory for an array of nmemb elements, each of sixe, and initializes it to zero.

3-array_range.c
Creates an array of integers containig all values from min to max inclusive.
