# Hash Tables in C

This project implements *hash table data structure in C* using *chaining for collision handling.* You will learn how to create, manage, and maniplulate hahs tables, similar to Python dictionaries but implemented from  scratch.

The project covers:

* Creating a hash table
* Implementing the **djb2** hash function
* Computing the index for a given key
* Adding, retrieving, and printing elements
* Deleting the hash table and freeing memory

---

## Files and Descriptions

0-hash_table_create.c
* Create a new hash table of a given size

1-djb2.c
* Implement the *djb2* hash function

2-key_index.c
* Compute the index of a key using djb2

3-hash_table_set.c
* Add or update a key/value pair

4-hash_table_get.c
* Retrieve the value associated with a key

5-hash_table_print.c
* Print the hash table in '{key: value, ...}' format

6-hash_table_delete.c
* Delete the hash table and free all memory
